## Analytics Infrastructure — Function App Deployment & Gold Table Testing
**Why:** Get analytics function app live and validate gold table refresh pipeline
**What:** Deployed function app to Azure. Ran end-to-end test on 5 gold tables (VenuesProposalRequests, VenuesAssignments, MetricsProposalRequests, MetricsEventPlans, DimVenues). Auth overhaul started — migrated from connection strings to Managed Identity for SQL access. Hit a snag with MI grant propagation delay (~5 min).
**Outcome:** Function app live, 23 gold tables confirmed, MI auth partially deployed
**On track?:** Focused — auth overhaul was planned but took longer than expected
**Lessons:** MI grant propagation can take 5+ minutes in Azure SQL. Always add a retry loop or wait step in deployment scripts. Connection string fallback is worth keeping until MI is battle-tested.
